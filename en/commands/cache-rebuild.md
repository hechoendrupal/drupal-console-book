# cache:rebuild
The **cache:rebuild** command Rebuild and clear all site caches.

**Usage:**
```
$ drupal cache:rebuild [arguments] 
$ cr  
```

## Available arguments
Argument | Details
---------|-------------
cache | Only clear a specific cache.

## Examples
* Rebuild all caches
```
$ drupal cr all
```
* Rebuild discovery cache
```
$ drupal cr discovery
```
