# views:disable
Disable a View

**Usage:**
```
drupal views:disable [arguments]
vd
```

## Available arguments
Argument | Details
---------|-------------
view-id | commands.views.debug.arguments.view-id

## Examples
* Disable content view
```
drupal views:disable  content
```
* Disable frontpage view
```
drupal views:disable  frontpage
```
