# rest:debug
The **rest:debug** command Display current rest resource for the application

**Использование:**
```
$ drupal rest:debug [arguments] [options] 
$ rede  
```

## Доступные опции
Опция | Описание
-------|-------------
--authorization | Rest resource status enabled | disabled

## Доступные параметры
Параметр | Описание
---------|-------------
resource-id | Rest ID
